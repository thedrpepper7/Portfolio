About Me

Hi, I'm Leon, an aspiring software developer currently expanding my knowledge with HTML and CSS. My interest in computers/programming started at a young age where I played computer games, appreciating how they worked and even thinking of a career in videogame making. This interest led to me doing computer science in school, learning Python, SQL and HTML, with more of a focus on Python having only entry level knowledge on HTML and SQL at the time. This learning provides me with basic understanding of programming concepts. By the end of my education in Python, I was able to build a password generator/ strength checker as part of a group. Now, I'm expanding my skills in HTML & CSS through the Scion Coalition Scheme, which forms the projects you'll find in this portfolio. My long-term aim is to become a versatile software developer with knowledge across various programming languages.
